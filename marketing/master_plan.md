## Production Checklist Per Video

1. [ ] Record screen with OBS (1080p, 60fps)
2. [ ] Import to CapCut
3. [ ] Add background music (CapCut library, low volume -15dB)
4. [ ] Add voiceover (record directly in CapCut or ElevenLabs free tier)
5. [ ] Add captions (CapCut auto-captions, edit for accuracy)
6. [ ] Add text overlays per script
7. [ ] Export as 1080×1920 MP4
8. [ ] Create thumbnail in Canva
9. [ ] Post: Twitter (native upload) + Instagram (Reels) + Facebook (Reels)
10. [ ] Add description with CTA + disclaimer + hashtags
